# Vibe Coding Workshop — Presentation

A self-contained HTML presentation built with the
[frontend-slides](https://github.com/jeffrey-groneberg/ghcp_sdk/tree/main/.copilot/skills/frontend-slides)
GitHub Copilot skill, deployed to GitHub Pages.

## Local preview

Open `index.html` in any modern browser, or serve the folder with a static server:

```powershell
python -m http.server 8000
```
